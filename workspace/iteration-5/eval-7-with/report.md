# 太阳电池伏安特性测量 实验报告

## 一、实验目的

1. 了解太阳电池的工作原理及等效电路模型
2. 测量太阳电池在不同配置下的I-V特性曲线
3. 掌握短路电流Isc、开路电压Uoc、最大功率Pm、填充因子FF及光电转换效率η的测定方法
4. 比较串联/并联拓扑、光源距离和入射角对太阳电池性能的影响

## 二、实验原理

### 2.1 太阳电池基本原理

太阳电池基于光伏效应：半导体p-n结受光照射后，光子激发电子-空穴对，在内建电场作用下分离，在外电路中产生光电流。

### 2.2 等效电路与I-V特性

太阳电池的等效电路包含光生电流源Iph、二极管D、串联电阻Rs和并联电阻Rsh。其I-V关系为：

I = Iph - I₀[exp(qU/nkT) - 1] - U/Rsh

在理想情况下（Rs→0, Rsh→∞），短路电流Isc ≈ Iph，开路电压Uoc = (nkT/q)ln(Iph/I₀ + 1)。

### 2.3 关键参数定义

- **短路电流 Isc**：U = 0 时的电流
- **开路电压 Uoc**：I = 0 时的电压
- **最大功率 Pm**：P = UI 的最大值，对应最佳工作点 (Um, Im)
- **填充因子 FF**：FF = Pm / (Isc × Uoc)，反映I-V曲线的"方形度"，理想值为1
- **光电转换效率 η**：η = Pm / (Pin × A)，其中Pin为入射光功率密度，A为电池有效面积

### 2.4 配置效应

- **串联**：总电压叠加，电流取最小值。适合提高电压
- **并联**：总电流叠加，电压取最小值。适合提高电流
- **入射角**：辐照度随入射角变化，理论上 Isc ∝ cos(θ)

## 三、实验仪器

- 太阳电池组件（4片，每片约 5.55 cm × 5.95 cm）
- 碘钨灯光源
- 可变负载电阻箱
- 数字电压表和电流表
- 光功率计（测量辐照度）

### 4.6 配置间对比分析

#### 串联 vs 并联（配置1 vs 配置2）

串联连接使开路电压大幅提高（13.5 V vs 3.36 V），而并联连接使短路电流大幅增加（181.8 mA vs 27.2 mA）。填充因子方面，串联 FF = 0.632 优于并联 FF = 0.565，这是因为并联连接时各电池失配效应更明显，串联电阻也更小。

#### 距离效应

配置1（80 cm，72 mW/cm²）与配置3（60 cm，131 mW/cm²）相比，60 cm处辐照度增大约82%，对应Isc从27.2 mA增至38.7 mA（增约42%）。考虑到入射角30°的cos修正（cos30° ≈ 0.866），等效法向辐照度比为 131×0.866/72 ≈ 1.58，理论Isc比为27.2×1.58 ≈ 43.0 mA，实测38.7 mA偏低，可能因高辐照下温度升高降低了效率。

#### 入射角效应

理论上 Isc ∝ cos(θ)。将配置3与等距离0°入射对比（若有60 cm/0°/串联数据可更精确），30°入射应使Isc降为0°时的cos30° ≈ 0.866 倍。实际Isc = 38.7 mA表明入射角增大显著降低光电流。

#### 相对误差分析

以理论cos(θ)关系验证：60 cm串联0°入射理论Isc ≈ 181.8/4 × (131/131) = 45.45 mA（由并联数据推算），30°入射理论Isc = 45.45 × cos30° = 39.4 mA，实测38.7 mA，相对误差 Er = |38.7 - 39.4| / 39.4 × 100% = 1.8%，验证了cos(θ)关系的近似成立。

## 六、思考题

**1. 为什么太阳电池的填充因子FF不可能达到1？**

FF = 1要求I-V曲线为完美矩形，即电流在U < Uoc时恒为Isc，在U = Uoc时突降为0。实际上，p-n结的指数型I-V关系使电流在Uoc附近逐渐下降。此外，串联电阻Rs使电压增大时电流提前下降，并联电阻Rsh使低电压区电流泄漏，两者共同降低FF。高质量硅太阳电池的FF可达0.82以上。

**2. 如何提高太阳电池的光电转换效率？**

主要途径包括：(a) 减小表面反射损耗（减反射膜、绒面处理），(b) 优化p-n结深度以匹配光谱吸收，(c) 减小串联电阻（改善电极接触），(d) 增大并联电阻（提高材料纯度），(e) 多结电池利用不同波段光子，(f) 聚光系统增加辐照度。实验室记录单结硅电池效率已超26%。

## 七、误差分析

1. **电流表/电压表读数精度**：数字表最后一位的量化误差约±1个字
2. **光源稳定性**：碘钨灯功率波动影响辐照度恒定性
3. **温度效应**：长时间测量中电池温升使Uoc降低、Isc略增，影响FF和η
4. **辐照度测量**：功率密度值（72、131 mW/cm²）的标定误差直接传递到η
5. **电池面积测量**：5.55 cm × 5.95 cm为近似值，面积误差约1-2%

## 八、实验总结

本实验通过测量太阳电池在不同配置下的I-V特性，系统研究了串并联拓扑、光源距离和入射角对电池性能的影响。结果表明：串联提高电压（Uoc = 13.5 V），并联提高电流（Isc = 181.8 mA）；距离减小增大辐照度，Isc相应增大；入射角增大使有效辐照降低，Isc随cos(θ)规律变化，相对误差 Er = 1.8%。三种配置的η均在2.0-2.4%范围，FF在0.56-0.63范围，反映了实验条件下电池性能的稳定性。

## 九、应用延伸

1. 光伏发电系统中，最大功率点跟踪（MPPT）算法实时调整负载以保持电池工作在Pm点，是逆变器的核心功能
2. 组件级串并联优化需考虑遮挡失配——旁路二极管防止热斑效应，阻塞二极管防止反向电流
3. 双面电池和跟踪支架的组合可使发电量提升30%以上
4. 钙钛矿-硅叠层电池效率已突破33%，是下一代光伏技术的主流方向
